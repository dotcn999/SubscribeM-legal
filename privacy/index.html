隐私政策

生效日期：2026-07-23 · 最近更新：2026-07-23 · 适用产品：SubscribeM（iOS）

中文 English

本政策说明我们如何处理与 SubscribeM 相关的信息。当前版本不运营用户账号， 不以开发者服务器收集您的订阅账单明细。内购交易由 Apple 处理。

1. 开发者与联系方式

产品名称：SubscribeM
Bundle ID：com.huazi.SubscribeM
联系邮箱：9366321@gmail.com

2. 我们收集哪些数据

2.1 您在设备上自行录入的内容（本地）

  • 订阅名称、价格、币种、周期、日期、备注、支付渠道标记、链接等账单记录；
  • 应用偏好（如基准货币、主题、提醒时间、安全锁开关等）。

上述数据默认保存在您的 iPhone/iPad 应用沙盒（例如 Documents 目录与 UserDefaults）中， 不会上传至我们运营的服务器（当前版本亦不提供 iCloud / CloudKit 自动同步）。

2.2 我们不主动收集的信息

  • 不收集姓名、电话、通讯录、精确地理位置、相册等；
  • 不接入广告或第三方追踪 SDK，不使用 IDFA 做广告追踪；
  • 无需注册登录即可使用核心功能。

3. 网络请求

当您主动或通过应用内功能更新汇率时，应用可能向公开汇率接口 （例如 exchangerate-api.com 等第三方服务）发起 HTTPS 请求，用于获取货币兑换参考汇率。

  • 请求中不包含您的订阅账单明细、备注或姓名等字段；
  • 第三方服务仍可能看到常规网络元数据（如 IP 地址、访问时间、User-Agent 等）；
  • 汇率结果仅供参考，不构成财务、投资或税务建议。

若您不更新在线汇率，应用可继续使用内置离线汇率表（可能不是最新）。

4. 应用内购买

SubscribeM Pro（按年自动续订、一次性买断等）通过 Apple App Store / StoreKit 完成。 支付、账单、退款与订阅管理由 Apple 根据其条款处理。我们无法从应用侧读取您的完整银行卡号等支付凭证。

5. 生物识别与安全锁（Pro）

若您开启 Face ID / Touch ID / 设备密码锁定，身份验证由 iOS 系统 LocalAuthentication 完成。应用不会存储您的生物识别模板或设备解锁密码。

6. 本地通知

扣费与试用提醒基于系统本地通知（UserNotifications），由您的设备本地调度。 通知正文来自您自行录入的订阅信息，不会因此将账单上传至我们的服务器。

7. 备份与导出（Pro）

若您使用 JSON / CSV 导出或通过系统分享面板发送备份，数据将按您选择的目标（文件、邮件、其他 App 等）传输。 请自行评估备份文件的保管风险；备份为明文结构化数据。

8. 数据保留与删除

本地数据随应用保留；卸载应用通常会删除应用沙盒内数据（以系统行为为准）。 因我们不托管您的账单账号数据，一般无需向我们提交“删除云端账单”的请求。 若您通过邮件向我们咨询，邮件通信将按常规邮件保留实践处理。

9. 儿童

本应用面向一般用户管理个人订阅支出，不面向 13 岁以下儿童定向收集个人信息。

10. 政策变更

我们可能更新本政策。重大变更将通过更新本页面“最近更新”日期，并在合理范围内在应用或本站提示。 继续使用应用即表示您知悉更新后的政策（适用法律另有规定的除外）。
