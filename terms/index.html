1. 服务说明

SubscribeM 是一款个人订阅账单记录与提醒辅助工具。您可以录入、整理各平台订阅信息，查看折算统计，并设置本地到期/试用提醒。

  • 本应用不会代替 Netflix、流媒体、SaaS 等服务商向您扣款；
  • 汇率、统计与提醒仅供参考，实际扣费以原服务商账单为准；
  • 您对录入信息的准确性与合法性负责。

2. 免费版与 SubscribeM Pro

免费版可使用基础订阅管理能力，活跃订阅数量存在上限（当前为最多 5 个活跃项目，以应用内实际限制为准）。

SubscribeM Pro 可通过 App 内购买解锁高级能力，例如（以应用内展示为准）：

  • 无限活跃订阅项目；
  • 高级统计分析图表；
  • Face ID / 密码安全锁；
  • JSON / CSV 备份导出与恢复。

3. 应用内购买与自动续订

Pro 可能提供例如：

  • 按年自动续订订阅（产品 ID 示例：com.huazi.SubscribeM.pro.yearly）；
  • 一次性买断（产品 ID 示例：com.huazi.SubscribeM.pro.lifetime）。

付款将记入您的 Apple ID 账户；价格以 App Store 展示的本地化价格为准。 若商品配置了免费试用，试用结束后除非您关闭自动续订，否则将按订阅周期续费。

自动续订：除非在当前周期结束前至少 24 小时关闭自动续订，否则订阅可能自动续订，并在续订前 24 小时内扣款（以 Apple 规则为准）。

管理与取消：在 iPhone「设置 → [您的姓名] → 订阅」中管理或取消。关闭自动续订后，已付费权益通常可用至当期结束。

恢复购买：可在应用内使用“恢复已购买项目”。
